Algunos Lenguajes de la web

HTML + CSS + Javascript

  • HTML5

    HTML5(HyperTextMakup Languaje, version 5) es la quinta version del Lenguaje básico de la World Wide Web, HTML.

  • CSS3

    Hoja de estilo de cascada o CSS(siglas en inglés de casch3ding style shets) e un lenguaje usado para definir y crear la presentación de un documento estructurado escrito en HTML.Usa la extensión css

  • Javascript

    JavaScript(abreviado comunmente como "JS") es un lenguaje de programación interpretado utilizando principalmente en el lado del clientes, implementando como parte de un navegador web permitiendo mejoras en la interfaz del usuario y páginas webs dinámicas. Usa la extensión .js
